1. 调试 ts 代码步骤
```
① 如果没有全局安装 typescript ，全局安装 npm install -g typescript 报错可以--force
② tsc --init 初始化
③ 创建 index.ts，写一行代码 `console.log(123)`
④ tsc --watch
⑤ 创建 index.html，引入 index.js，使用 Live Server 打开，打开 开发者工具 控制台
```
